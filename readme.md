# Christmas

Adds snow and presents for the [Minetest Engine](https://www.minetest.net).

## Chatcommands

- `/christmas_toggle` Toggles Christmas mode.

## Nodes

- `christmas:present_spawner` spawns presents around itself on `default:dirt_with_snow`.
